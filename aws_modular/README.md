# Terraform 모듈화

## 개요

VPC, EC2, ALB 등 반복되는 인프라 구성을 재사용 가능한 모듈로 분리하는 패턴입니다. 여러 환경(dev/staging/prod)에서 동일한 모듈을 재사용할 수 있습니다.

## 디렉토리 구조

```
aws_modular/
├── modules/
│   ├── vpc/          # VPC, 서브넷, IGW, NAT 모듈
│   ├── ec2/          # EC2, 보안 그룹 모듈
│   └── alb/          # ALB, 타겟 그룹, 리스너 모듈
├── envs/
│   ├── dev/          # 개발 환경
│   └── prod/         # 운영 환경
```

## 특징

- 모듈 한 번 작성 후 여러 환경에서 재사용
- 환경별 변수만 다르게 주입 (인스턴스 타입, 서브넷 크기 등)
- 인프라 표준화 및 일관성 유지
